Projects
Avengers Clicky Game

A simple game where the use clicks on an image, and tries to click on all of the images only once. The images change positions after each click. The game resets if you hit the same image twice, and the score starts over.


Does the Dog Live?

This application is for dog lovers who are not sure if they want to see a movie that contains a dog if the fate of the dog is unknown. Users can check the list of movies already provided which state if the dog lives or dies, and users can add movies themselves to contribute to the database.


Burger App

This application allows the user to click on a burger to “devour” it and add new burgers to the list. This app utilizes MySQL database, express, and handlebars.


Skyrim Guide

This is a simple guide for beginner players of the video game Elder Scrolls V: Skyrim. It includes an abreviated character guide, easy keyword search for images and links, and an area map with points of interest.


Pokemon Giphy

This little application allows you to select various buttons to pull up images/giphy of Pokemon. Clicking the images will toggle them from moving to still. You can also add new Pokemon by creating a new button with your Pokemon of choice.


Friend Finder

This application directs you to answer 10 silly questions which will then select the closest matching friend from the database. The six original hard coded friends are some of the Avengers, which is why the questions are silly and pertain to some of them.
